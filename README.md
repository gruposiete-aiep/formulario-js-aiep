## Descripción

Formulario web desarrollado con HTML, CSS y JavaScript que incluye validación de campos, manejo de eventos DOM y simulación de envío mediante promesas. Link del dormulario: https://gruposiete-aiep.github.io/formulario-js-aiep/

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)

## Funcionalidades

- Validación de campos obligatorios (nombre, correo, mensaje)
- Validación de formato de correo con API nativa HTML5
- Eventos DOM: `blur` y `submit`
- Simulación de envío con `Promise` y espera de 2 segundos
- Control de errores visible al usuario (éxito y fallo)
